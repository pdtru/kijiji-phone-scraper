## Getting Started

<br>

### Prerequisites

- Install node on your local machine: [Here](https://nodejs.org/en/download/current)

## Usage

- Install dependecnies by running the following command in the terminal:

```sh
npm install
```

<br>

- Run the program with the following command:

```sh
node index.js
```
